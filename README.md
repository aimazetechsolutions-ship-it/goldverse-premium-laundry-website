# Goldverse Premium Laundry Website

Standalone static website for Goldverse Premium Laundry (Pvt) Limited.

## Project

- `index.html` - Website markup
- `styles.css` - Responsive styling
- `script.js` - Header and mobile navigation behavior
- `assets/` - Local real photo assets used by the website

## Preview

Open `index.html` directly in a browser, or serve this folder with any static web server.
